# Process Data Telemetry 🚀

> Obtención de datos de telemetría de videojuegos de carrera (simracing) inicialmente pensado para F1 2O21 y en el cuál se pensará más adelante para escalarlo a iRacing

## 🌟 Características

- ⚡ Procesamiento de datos postcarrera.
- 📊 Visualización gráfica interactiva.
- 🔒 Autenticación segura y validación estricta de datos.
- 🛠️ Arquitectura limpia y modular.

## 🛠️ Tecnologías Utilizadas

- **Otros:** UDP Sockets con python
- **Backend:** PHP 8.2 / Laravel 13
- **Frontend:** Vue 3
- **Base de datos:** PostgreSQL
